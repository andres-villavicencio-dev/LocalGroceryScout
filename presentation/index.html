The Fortress

Three-tier defense architecture

Layer 1: Client Validation
AllowList patterns block prompt injection, XSS, SQL injection
searchQuery: 100 chars, alphanumeric + basic punctuation
barcode: 8-14 digits only
Layer 2: Service Validation
Pre-API call validation in geminiService.ts and firestoreService.ts
Response sanitization removes dangerous HTML/JS
Error logging for security monitoring
Layer 3: Firebase Security Rules
Field-level access control (server-authoritative isPro flag)
Max 20 shopping lists per user, 100 items per list
1MB document size enforcement
